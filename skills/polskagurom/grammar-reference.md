## 1. Przypadki — kiedy używać

| # | Przypadek | Pytania | Typowe użycie |
|---|---|---|---|
| 1 | Mianownik (M.) | kto? co? | podmiot |
| 2 | Dopełniacz (D.) | kogo? czego? | brak, posiadanie, przeczenie |
| 3 | Celownik (C.) | komu? czemu? | beneficjent |
| 4 | Biernik (B.) | kogo? co? | dopełnienie bliższe w zd. twierdzących |
| 5 | Narzędnik (N.) | (z) kim? (z) czym? | narzędzie, orzecznik („jest lekarzem") |
| 6 | Miejscownik (Ms.) | o kim? o czym? | po: w, na, o, po, przy |
| 7 | Wołacz (W.) | hej! | zwracanie się („Aniu!") |

### Biernik vs dopełniacz — reguła przeczenia

**Twierdzenie → biernik. Przeczenie → dopełniacz.**

| Twierdzenie (B.) | Przeczenie (D.) |
|---|---|
| Mam czas. | Nie mam czasu. |
| Widzę Anię. | Nie widzę Ani. |
| Kupiłem chleb. | Nie kupiłem chleba. |
| Lubię kawę. | Nie lubię kawy. |

> Najczęstszy błąd AI: zostawia biernik po przeczeniu.

### Męskoosobowość (l.mn.)

- **Męskoosobowy** (mężczyźni): B. l.mn. = D. l.mn. → "widzę studentów"
- **Niemęskoosobowy** (kobiety, dzieci, zwierzęta, rzeczy): B. l.mn. = M. l.mn. → "widzę koty, kobiety, dzieci"

## 3. Aspekt — dokonany vs niedokonany

| Niedokonany (proces, powtarzalność) | Dokonany (zakończenie, jednorazowość) |
|---|---|
| robić | zrobić |
| pisać | napisać |
| czytać | przeczytać |
| kupować | kupić |
| brać | wziąć |
| mówić | powiedzieć |
| widzieć | zobaczyć |
| wracać | wrócić |
| zaczynać | zacząć |
| kończyć | skończyć |

### Typowe błędy

1. **Czas teraźniejszy od dokonanego = przyszłość!** "Napiszę" = future, nie present.
2. **Brak teraźniejszości u dokonanych** — nie ma `*napisuję`.
3. **"Zaczynać/kończyć" + bezokolicznik → tylko niedokonany:** "zacznij pisać" ✓, ~~"zacznij napisać"~~ ✗.
4. **"Lubić/woleć/umieć" + bezokolicznik → tylko niedokonany:** "lubię czytać" ✓.

## 5. Liczebniki

### Składnia z rzeczownikiem

| Liczebnik | Składnia | Przykład |
|---|---|---|
| 1 | M. l.poj. | jeden student |
| 2, 3, 4 (i 22, 23, 24…) | M. l.mn. | dwa koty, trzy książki |
| 5+ (oraz 21, 25–30, 100…) | D. l.mn. | pięć kotów, dziesięć książek |

### Męskoosobowość zmienia formę

| Niemęskoosobowy | Męskoosobowy |
|---|---|
| dwa psy / dwie kobiety | dwaj mężczyźni / dwóch mężczyzn |
| pięć kobiet | pięciu mężczyzn |
| dziesięć książek | dziesięciu chłopców |

### Liczebniki zbiorowe (-oje, -oro) — kiedy obowiązkowe

1. **Grupy mieszane**: dwoje studentów, pięcioro nauczycieli
2. **Istoty niedorosłe**: dwoje dzieci, troje cieląt
3. **Pluralia tantum**: dwoje drzwi, troje skrzypiec
4. **Para rodziców**: oboje rodzice

Błąd: ~~"dwa dzieci"~~ → **dwoje dzieci**.

### Czasownik z liczebnikiem 5+

Rodzaj nijaki l.poj.: **"Pięć osób przyszło"**, **"Dwoje dzieci bawiło się"** — NIE "przyszły".

## 7. Interpunkcja — kluczowe reguły

### Przecinek PRZED — zawsze

Przed: **że, który, jeśli, jeżeli, gdy, kiedy, ponieważ, bo, gdyż, aby, żeby, by, choć, chociaż, mimo że, dopóki, jak (porównanie), niż**

- Wiem, że przyjdziesz.
- Człowiek, który czeka.
- Zadzwonię, jeśli zdążę.
- Pracuję, żeby zarobić.

### Przecinek przed "i, oraz, lub, albo, ani, czy"

**Reguła główna: NIE stawiamy.** "Kupiłem chleb i masło."

**Wyjątki — stawiamy:**
1. **Spójnik powtórzony**: "Kupiłem **i** chleb, **i** masło, **i** ser."
2. **Po wtrąceniu/zdaniu podrzędnym kończącym się przed "i"**: "Pracuję w firmie, która produkuje meble, i lubię to."

### Imiesłowy

- **Przysłówkowy (-ąc, -wszy)** → zawsze przecinek: "Idąc do domu, spotkałem Anię."
- **Przymiotnikowy (-ący, -ony)** bez rozbudowania → bez przecinka.

### Znaki kreskowe

| Znak | Wygląd | Użycie |
|---|---|---|
| łącznik | - | wewnątrz wyrazów: biało-czerwony, e-mail |
| półpauza | – | zakresy: 10–15, 1939–1945 |
| myślnik (pauza) | — | wtrącenia ze spacjami: "To — moim zdaniem — błąd" |

### Cudzysłowy

- Polski: **„xxx"** (otwierający dolny, zamykający górny)
- Drugiego stopnia: **»xxx«**
- ~~"xxx"~~ proste — błąd typograficzny

## 8. Pisownia łączna i rozdzielna

### "nie"

| Część mowy | Reguła | Przykład |
|---|---|---|
| czasowniki | **rozdzielnie** | nie wiem, nie chcę |
| rzeczowniki | **łącznie** | niepokój, nieprawda |
| przymiotniki (st. równy) | **łącznie** | nieładny, niedrogi |
| przymiotniki (st. wyższy/najwyższy) | **rozdzielnie** | nie lepszy, nie najlepszy |
| przysłówki od przym. (st. równy) | **łącznie** | niedobrze, nieładnie |
| przysłówki (st. wyższy/najwyższy) | **rozdzielnie** | nie lepiej, nie najlepiej |
| przysłówki niepochodne | **rozdzielnie** | nie bardzo, nie zawsze, nie tu |
| liczebniki | **rozdzielnie** | nie pięć (wyj.: niejeden, niewiele — łącznie) |
| zaimki | **rozdzielnie** | nie ja, nie ten, nie mój |
| imiesłowy przymiotnikowe | **łącznie** | niezrobiony, niepiszący |
| imiesłowy przysłówkowe | **rozdzielnie** | nie pisząc, nie zrobiwszy |

**Przeciwstawienie:** zawsze rozdzielnie → "nie ładny, lecz piękny".

### "by / żeby / aby"

- z czasownikiem w czasie przeszłym → **łącznie**: zrobiłbym, poszedłbyś
- ze spójnikiem → **łącznie**: gdyby, choćby, jakby
- "żeby", "aby", "oby" → zawsze **łącznie**

### "naj-" → zawsze łącznie

## 9. Wielkie / małe litery

### Wielką literą
- Imiona, nazwiska, geo: Anna, Warszawa, Tatry
- Mieszkańcy: Polak, Niemiec, Europejczyk
- Święta: Boże Narodzenie, Wielkanoc, Dzień Matki
- Instytucje (każdy człon znaczący): Uniwersytet Warszawski
- Tytuły dzieł — TYLKO pierwsze słowo: „Pan Tadeusz"
- Formy grzecznościowe w listach: Ty, Tobie, Państwo, Pan

### Małą literą
- Przymiotniki od nazw własnych: polski, niemiecki
- Stanowiska: prezydent, minister, dyrektor
- Religie: katolicyzm, islam
- Kierunki świata jako kierunki: północ, południe
- Dni tygodnia, miesiące: poniedziałek, styczeń
- Waluty, jednostki: złoty, metr

### "który" — uzgodnienie

Rodzaj i liczba z rzeczownika, przypadek z funkcji w zdaniu podrzędnym.
- "Książka, **którą** czytam" (B.)
- "Człowiek, **któremu** ufam" (C.)
- "Dom, **w którym** mieszkam" (Ms.)

### "swój" vs "jego/jej/ich"

- **swój** → posiadacz = podmiot zdania
- **jego/jej/ich** → posiadacz ≠ podmiot

"Anna kocha **swoje** dzieci" (Anny dzieci) vs "Anna kocha **jej** dzieci" (czyjeś inne).
